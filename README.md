# Tic Tac Toe Game

## Description
A console-based Tic Tac Toe game developed in C++ using arrays, loops, and conditional statements.

## Features
- Dynamic Board Display
- Two Player Mode
- Win Detection
- Draw Detection
- Replay Option

## Technologies
- C++
- Arrays
- Loops
- Conditional Statements

## How to Run

Compile:

g++ main.cpp -o game

Run:

./game
